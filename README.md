# 個人的に好きな日本の歌を韓国語で翻訳したサイトです。

ボタンで歌手を選んだ後、メニューから曲を選択すると歌詞が
</br>
＜日本語歌詞＞
</br>
＜韓国語で読み方＞
</br>
＜韓国語で翻訳＞
</br>
の形で翻訳されております。
